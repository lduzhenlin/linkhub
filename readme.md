
### LinkHub 介绍

### 项目介绍

LinkHub 是一款链接管理工具，支持PC端和手机端访问，可以帮助用户管理分散的链接。对于普通用户来说，无论是管理日常生活中的各种网站链接、社交媒体链接，还是整理学习资料、娱乐资源等链接，LinkHub 都能提供便捷的管理方式，使个人的在线资源更加有序，方便。

### 内置功能

------

- 分组管理：分组的增删改查功能

- 链接管理：链接的增删改查功能。

### 运行步骤
```
1.克隆项目

2.导入数据库 ：新建数据库名：zeus_nav_dev，然后将项目目录doc/zeus_nav_dev.sql导入到数据库中，

3.配置数据库密码： 修改application-dev.yml中的数据库配置，修改成你自己的数据库账号和密码

4.运行项目

5.登录 用户名：1 密码：1
```
